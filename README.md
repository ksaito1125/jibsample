# Spring boot applicationサンプル

## ビルドと実行

下記のコマンドでビルドします。

```
mvn package
```

下記のコマンドでアプリケーションを起動します。

```
java -jar target/gs-spring-boot-0.1.0.jar
```

下記のコマンドでアプリケーションにアクセスできます。

```
$ curl localhost:8080
Greetings from Spring Boot!
$
```
